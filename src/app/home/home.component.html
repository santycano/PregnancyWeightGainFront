Grupo latinoamericano de ganancia de peso gestacional de adolescentes

De que se trata este proyecto?

Este proyecto desarrollo curvas para la vigilancia de la gancia de peso de embarazadas adolescentes Latinoamericanas, una para cada categoria de indice de masa corporal pregestacional .

Paises participantes

Colombia, Brasil, Uruguay, Panama, Argentina, Mexico, Paraguay, Chile, Peru.
